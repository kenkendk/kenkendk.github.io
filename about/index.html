About this site

What is HexaD?

HexaD is the organization name, that I used for developing opensource applications. This website contains links to the projects that I participate in, and my CV.

Current projects

I currently work on these open source projects:

  1. Duplicati, a backup system with multiple storage options and a simple user interface
  2. Bohrium, a runtime environment for efficiently executing vectorized applications
  3. SME, a simulation tool suitable for developing hardware models
  4. CoCoL, a library for thread-safe communication
  5. Ceen.Httpd, A tiny compliant HTTP server written entirely in C# with async/await all the way.
